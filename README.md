# Nova Landing

Nova를 Claude Code용 AI Agent Ops 프레임워크로 소개하는 정적 랜딩 사이트입니다.

## Stack

- Next.js 16 App Router
- TypeScript
- Tailwind CSS v4
- Motion
- pnpm

## Local Development

```bash
pnpm dev -p 4100
pnpm build
```

## Deploy

GitHub Pages 배포는 `main` 브랜치 push 후 GitHub Actions가 정적 빌드를 실행합니다.

Production URL: https://jay-swk.github.io/nova-landing/
